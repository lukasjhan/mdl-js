# Type definitions for mdl

This package provides type definitions for the mdoc/mdl(ISO/IEC 18013-5).

## Supported Platforms

- Node.js
- Browser
- React Native

## Installation

```bash
npm install @m-doc/types
```

```bash
yarn install @m-doc/types
```

```bash
pnpm install @m-doc/types
```
