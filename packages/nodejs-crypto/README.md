# Node.js crypto for mdl

This package provides crypto functions for the mdoc/mdl(ISO/IEC 18013-5).

## Supported Platforms

- Node.js

## Installation

```bash
npm install @m-doc/nodejs-crypto
```

```bash
yarn install @m-doc/nodejs-crypto
```

```bash
pnpm install @m-doc/nodejs-crypto
```
